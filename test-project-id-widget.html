Webhook Request Format

This is the data the widget sends to your Make.com webhook for each action.

Voice Call Request

{
  "project_id": "ZWQ4VZV",
  "mode": "voice",
  "dynamic_variables": {
    "company_name": "Acme Corp"
  }
}

Chat Message Request

{
  "project_id": "ZWQ4VZV",
  "mode": "chat",
  "action": "send_message",
  "chat_id": "chat_abc123",
  "message": "Hello!"
}

Embed Code

Paste this into your website HTML just before the closing </body> tag.

<script src="https://YOUR-HOSTING-URL/fluvio-universal-widget.js"
        data-webhook="https://hook.us2.make.com/your-webhook-url"
        data-project-id="YOUR-PROJECT-ID"
        data-mode="dual"
        data-default-mode="voice"></script>
