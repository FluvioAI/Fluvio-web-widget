Your Embed Code

Copy this code and paste it into your website's HTML, just before the closing </body> tag. Replace the URL with your hosted widget URL.

<!-- Voice + Chat Widget -->
<script src="https://YOUR-HOSTING-URL/fluvio-universal-widget.js"
        data-webhook="https://hook.us2.make.com/your-webhook-url"
        data-project-id="YOUR-PROJECT-ID"
        data-mode="dual"
        data-default-mode="voice"
        data-title="AI Assistant"
        data-subtitle="Voice & Chat Support"></script>

<!-- Voice Only -->
<script src="https://YOUR-HOSTING-URL/fluvio-universal-widget.js"
        data-webhook="https://hook.us2.make.com/your-webhook-url"
        data-project-id="YOUR-PROJECT-ID"
        data-mode="voice"></script>

<!-- Chat Only -->
<script src="https://YOUR-HOSTING-URL/fluvio-universal-widget.js"
        data-webhook="https://hook.us2.make.com/your-webhook-url"
        data-project-id="YOUR-PROJECT-ID"
        data-mode="chat"></script>

Configuration Options

All available options for the widget script tag.

Required

  • data-webhook — Make.com webhook URL
  • data-project-id — Your Fluvio project ID

Mode

  • data-mode — dual, voice, or chat
  • data-default-mode — voice or chat

Appearance

  • data-color — Brand color (hex)
  • data-title — Widget header title
  • data-subtitle — Header subtitle
  • data-position — bottom-right, bottom-left, top-right, top-left

Personalization

  • data-agent-name — AI agent name
  • data-company-name — Company name
  • data-greeting — Opening chat message
